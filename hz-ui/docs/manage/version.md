## 版本说明

本文档记录 HZ-MANAGE 的主要版本更新内容。

---

## v1.0.0

### 新增功能

- **CMDB 资产平台**
  - 模型管理：自定义模型、字段、字段类型
  - 资源实例：资产的增删改查
  - 资源检索：多条件组合查询
  - 关联关系：模型间的拓扑关联
  - 资产审计：变更历史追溯

- **管控平台**
  - 节点管理：服务器、设备管理
  - 代理配置：跳板机、HTTP 代理
  - 批量执行：基于 Ansible 的命令执行

- **日志应用**
  - 日志检索：Loki 日志查询
  - 流程日志：跨环节日志关联分析
  - 环节配置：自定义关联规则

- **系统管理**
  - 用户管理：用户、用户组管理
  - 角色权限：RBAC 权限控制
  - 菜单管理：动态菜单配置

- **门户功能**
  - 门户收藏夹：个性化首页
  - 小工具：二维码、JSON 格式化、时间戳转换

### 技术特性

- 前后端分离架构
- WebSocket 实时通信
- 国密算法加密
- Docker 容器化部署
