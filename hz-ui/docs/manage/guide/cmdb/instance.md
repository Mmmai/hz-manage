title: 资源实例
---

## 概述

资源实例管理是 CMDB 的数据维护页面，用于创建、编辑、删除资产实例。

### 主要功能

- **创建实例**：根据模型定义创建新的资产实例
- **编辑实例**：修改已有资产的属性值
- **删除实例**：删除不再需要的资产
- **批量导入**：通过 Excel 批量导入资产数据
- **批量导出**：导出资产数据为 Excel

### 操作说明

#### 创建实例

1. 点击"新增"按钮
2. 选择模型类型
3. 填写必填字段（标 * 号）
4. 保存提交

#### 导入导出

**导入格式要求**：
- 第一行为字段名称
- 数据需符合字段类型约束
- 引用字段需确保目标实例存在

**密码字段**：
- 导入时支持明文
- 显示时自动脱敏
- 支持显示密码功能（需权限）

### 权限控制

按钮可见性基于用户权限配置：
- **导入**：需导入权限
- **导出**：需导出权限
- **显示密码**：需显示密码权限
