title: 模型管理
---

## 概述

模型管理是 CMDB 的核心功能，基于模型驱动的资产管理理念，支持用户自定义模型、字段、类型等。

### 模型概念

模型（Model）是资产的抽象定义，描述了一类资产的共同属性。例如：

- **服务器模型**：包含 IP、主机名、操作系统、CPU、内存等字段
- **数据库模型**：包含端口、版本、字符集等字段

### 功能说明

#### 模型配置

- 创建自定义模型
- 配置模型字段
- 设置字段分组
- 定义字段类型

#### 字段类型支持

| 类型 | 说明 | 示例 |
|------|------|------|
| 文本 | 单行文本 | 主机名 |
| 数字 | 整数/小数 | CPU 核数 |
| 密码 | 国密算法加密 | 数据库密码 |
| 日期 | 日期时间 | 创建时间 |
| 选择 | 下拉选择 | 操作系统 |
| 引用 | 关联其他模型 | 所属机房 |

### 使用场景

1. **定义新资产类型**：根据业务需求创建自定义模型
2. **字段约束**：通过校验规则确保数据质量
3. **关联关系**：建立模型间的引用关系

### 最佳实践

- 合理规划字段分组，便于维护
- 使用引用字段建立资产关联
- 启用校验规则防止错误数据
