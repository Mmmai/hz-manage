title: 关联关系
---

## 功能概述

关联关系定义模型之间的引用关系，支持建立资产间的拓扑连接。

### 关系类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 一对一 | 单向引用 | 服务器 → BMC IP |
| 一对多 | 父子关系 | 机房 → 服务器 |
| 多对多 | 双向关联 | 应用 ↔ 服务器 |

### 配置说明

1. 定义源模型和目标模型
2. 选择关系类型
3. 设置关系名称
4. 配置约束规则

### 关系视图

系统提供可视化拓扑图，展示：
- 节点：资产实例
- 连线：关联关系
- 方向：引用方向

### 使用场景

- **应用依赖**：应用依赖的数据库、中间件
- **网络拓扑**：交换机、路由器连接关系
- **物理位置**：机柜、设备的物理放置
