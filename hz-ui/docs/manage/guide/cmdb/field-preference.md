## 概述

字段偏好（Field Preference）允许用户自定义实例列表中字段的显示和隐藏，提供个性化的数据展示体验。

### 功能价值

- **个性化展示**：用户可按需显示字段
- **提高效率**：只关注需要的字段
- **减少干扰**：隐藏不常用的字段
- **灵活配置**：随时调整显示设置

## 字段偏好概念

### 偏好作用范围

字段偏好是用户个人的设置：

- **用户级**：每个用户有独立的偏好设置
- **模型级**：每个模型的偏好设置独立
- **持久化**：偏好设置会被保存，下次登录自动加载

::: tip 偏好不影响权限
字段偏好只影响显示，不影响权限：
- 隐藏的字段仍然存在于数据库
- 在实例详情页仍可查看
- 编辑时仍可修改
:::

### 默认显示

未设置偏好的情况下，系统显示默认字段：
- 实例名称（必显示）
- 常用字段（如 IP、主机名等）
- 系统根据字段类型和重要性决定

## 设置字段偏好

### 显示/隐藏字段

1. 进入 **资源实例** 页面
2. 点击 **列设置** 或 **字段偏好** 按钮
3. 在弹出的对话框中：
   - 勾选要显示的字段
   - 取消勾选要隐藏的字段
4. 点击 **保存**

::: tip 字段选择建议
- 常用字段：建议显示（如主机名、IP）
- 业务字段：根据需要显示
- 系统字段：通常隐藏（如创建时间、更新时间）
:::

### 字段排序

1. 进入字段偏好设置
2. 拖拽字段调整顺序
3. 点击 **保存**

字段排序影响列表中列的显示顺序。

## 重置偏好

### 重置为默认

1. 进入字段偏好设置
2. 点击 **重置为默认** 按钮
3. 确认重置操作

重置后：
- 所有字段恢复为默认显示状态
- 字段顺序恢复默认
- 个人偏好设置被清除

::: tip 重置的影响
重置只会影响当前模型：
- 其他模型的偏好不受影响
- 其他用户的偏好不受影响
:::

## 偏好使用场景

### 场景一：运维人员视图

**需求**：运维人员关注服务器的配置信息。

**显示字段**：
- 实例名称
- 主机名
- IP 地址
- CPU 核数
- 内存大小
- 操作系统

**隐藏字段**：
- 描述
- 标签
- 创建时间
- 更新时间

### 场景二：业务人员视图

**需求**：业务人员关注应用的负责人和状态。

**显示字段**：
- 实例名称
- 应用名称
- 负责人
- 状态
- 环境
- 业务线

**隐藏字段**：
- 技术配置
- 部署信息
- 系统字段

### 场景三：网络管理员视图

**需求**：网络管理员关注网络设备的连接信息。

**显示字段**：
- 设备名称
- IP 地址
- 子网掩码
- 网关
- 连接端口
- VLAN

**隐藏字段**：
- 设备描述
- 维护信息
- 系统字段

## 最佳实践

### 偏好设置建议

1. **显示常用字段**
   - 识别最常查看的字段
   - 设置为显示状态
   - 避免显示过多字段

2. **隐藏不常用字段**
   - 隐藏很少查看的字段
   - 减少界面干扰
   - 需要时在详情页查看

3. **合理排序**
   - 重要字段排在前面
   - 相关字段放在一起
   - 便于快速浏览

4. **定期调整**
   - 根据工作变化调整偏好
   - 删除不再需要的字段显示
   - 添加新的常用字段

### 团队协作建议

1. **共享偏好模板**
   - 团队内约定常用字段
   - 新成员按模板设置
   - 保持团队一致性

2. **文档记录**
   - 记录常用偏好配置
   - 便于新成员快速上手
   - 作为团队规范

3. **定期评审**
   - 定期评审字段偏好
   - 根据业务变化调整
   - 优化工作效率

## 常见问题

### Q: 字段偏好对所有用户生效吗？

A: 不。字段偏好是用户个人的设置：
- 每个用户有独立的偏好
- 互不影响
- 登录后自动加载个人偏好

### Q: 字段偏好会影响导出吗？

A: 不会。导出功能：
- 导出所有字段（不受偏好影响）
- 或导出当前显示的字段（取决于配置）
- 具体行为取决于导出功能实现

### Q: 隐藏的字段还能查看吗？

A: 可以。隐藏字段：
- 只是在列表中不显示
- 在实例详情页仍可查看
- 编辑时仍可修改
- 查询时仍可使用

### Q: 如何批量设置字段偏好？

A: 当前版本不支持批量设置：
- 每个用户需要单独设置
- 每个模型的偏好单独设置
- 可以通过分享配置模板指导用户设置

### Q: 字段偏好会随系统更新丢失吗？

A: 不会。字段偏好：
- 存储在数据库中
- 系统更新不影响偏好
- 除非数据库被清空

### Q: 管理员可以设置默认偏好吗？

A: 取决于系统实现：
- 某些系统支持配置默认字段
- 或通过代码配置默认显示字段
- 新用户使用默认偏好

### Q: 字段偏好和权限有什么关系？

A: 两者独立：
- **字段偏好**：控制字段显示/隐藏
- **权限控制**：控制字段访问权限

即使隐藏字段，有权限的用户仍可在详情页查看。
