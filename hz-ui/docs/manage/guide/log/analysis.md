title: 流程日志
---

## 功能概述

流程日志支持跨多个环节的日志关联分析，适用于追踪业务全流程。

### 应用场景

- **请求追踪**：从接入到后端的完整链路
- **故障排查**：定位问题发生的环节
- **性能分析**：分析各环节耗时

### 环节配置

预定义关联环节：
- 接入层（Nginx）
- 应用层（业务日志）
- 数据层（数据库慢查询）

### 使用说明

1. 选择分析任务
2. 输入关联标识（如 trace_id、request_id）
3. 查看各环节日志
4. 分析异常点

### 可视化

- 时间线视图
- 环节耗时统计
- 调用拓扑图
