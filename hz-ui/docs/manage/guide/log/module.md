title: 环节配置
---

## 功能概述

环节配置用于定义流程日志分析的关联环节，配置日志源和关联规则。

### 配置步骤

#### 1. 添加环节

- 定义环节名称
- 选择日志源类型（Loki/数据库/文件）
- 配置连接参数

#### 2. 配置关联规则

- 选择关联字段
- 定义匹配条件
- 设置时间窗口

#### 3. 启用环节

保存配置后启用，即可在流程日志中使用。

### 配置示例

```
环节：用户登录
├── 日志源：应用日志
├── 关联字段：user_id
└── 时间窗口：5分钟
```

### 最佳实践

- 合理设置时间窗口
- 使用唯一标识关联
- 避免循环依赖
