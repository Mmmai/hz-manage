## 功能概述

日志检索基于 Loki 日志系统，提供高效的日志查询和分析能力。

### 查询语法

#### 基础查询

```
{job="mysql"}           # 按标签筛选
{job="mysql"} |= "error"  # 包含关键字
{job="mysql"} != "debug"  # 排除关键字
```

#### 正则查询

```
{job="mysql"} |=~ "ERROR.*timeout"
```

#### 时间范围

- 最近 15 分钟
- 最近 1 小时
- 最近 24 小时
- 自定义范围

### 功能特点

- **实时查询**：秒级响应
- **标签索引**：按标签快速筛选
- **高亮显示**：匹配内容高亮
- **导出日志**：导出查询结果

### 使用建议

- 使用标签缩小查询范围
- 关键词尽量精确
- 设置合理的时间范围
