## 功能概述

节点管理用于管理服务器、设备等管控节点，支持远程执行命令和脚本。

### 节点类型

- **服务器**：物理机、虚拟机
- **网络设备**：交换机、路由器
- **存储设备**：存储阵列、NAS

### 主要功能

#### 节点注册

- 添加节点信息
- 配置连接方式
- 设置节点分组

#### 连接配置

- **SSH 连接**：用户名/密码或密钥
- **代理连接**：通过代理服务器中转
- **连接测试**：验证配置正确性

#### 节点状态

实时显示节点状态：
- 在线/离线
- 响应时间
- 资源使用率

### 使用说明

1. 添加节点
2. 配置连接参数
3. 测试连接
4. 分配到业务分组
