# 系统简介

HZ-MANAGE 是中心自研的企业级运维管理平台，响应实际运维需求，打造适用于现场维护的一体化运维体系。

### 核心特性

- **配置管理**：基于 CMDB 设计理念，实现自定义模型、字段、类型的灵活资产配置
- **权限控制**：标准 RBAC 权限设计，支持菜单、按钮级别的细粒度权限控制
- **日志平台**：采用 Loki 架构，实现统一日志检索和关联分析
- **节点管控**：支持批量命令执行，高效管理大规模节点
- **门户定制**：个性化门户收藏夹，按需分组管理常用功能

---

## 功能概览

| 模块 | 说明 |
|------|------|
| **门户收藏夹** | 将常用页面加入门户，支持分组管理和快速访问 |
| **资产配置** | 基于 CMDB 的模型驱动资产管理，支持自定义模型和关联关系 |
| **管控平台** | 节点管理、代理配置，支持远程命令执行 |
| **作业平台** | 基于 Ansible 的批量任务执行 |
| **日志应用** | Loki 日志检索、流程日志关联分析 |
| **系统管理** | 用户、角色、权限、菜单管理 |

## 技术栈

- **前端**：Vue 3 + Vite + Element Plus + Ant Design Vue
- **后端**：Python + Django + Django REST Framework
- **缓存**：Redis
- **日志**：Loki
- **任务**：Celery + Ansible
